# Evidencia 02: Comunicado interno institucional simulado

## Descripción general

Este proyecto consiste en una página web estática desarrollada con HTML y CSS, diseñada para simular un portal interno de comunicación académica de una institución educativa ficticia llamada **Academia de Danza MX**.

La evidencia fue construida con el objetivo de demostrar habilidades de **integración de contenido web**, organización clara de información, jerarquía visual, uso de navegación interna y adaptación responsive para distintos tamaños de pantalla.

El contenido presentado simula un comunicado dirigido a coordinación académica y personal docente, con información relacionada con actividades del periodo, asignaturas, calendario general, fechas clave, lineamientos de uso de espacios y medios de contacto institucional.

---

## Objetivo del proyecto

El propósito principal de esta evidencia es mostrar la capacidad de desarrollar una página informativa clara, ordenada y visualmente coherente, enfocada en la presentación de contenido para usuarios dentro de un entorno institucional.

Este proyecto busca demostrar el manejo de elementos fundamentales de HTML y CSS aplicados a una necesidad concreta de comunicación web, tales como:

- organización semántica del contenido
- uso de títulos, párrafos, listas y enlaces
- jerarquía visual para facilitar la lectura
- navegación principal y navegación interna por secciones
- integración de recursos visuales
- diseño responsive para escritorio, tablet y móvil

---

## Función que se pretende con este proyecto

La función de esta página es simular un espacio web donde una institución puede publicar comunicados internos de forma clara, accesible y ordenada para su comunidad académica.

En este caso, el proyecto representa un portal que podría utilizarse para comunicar:

- organización académica del periodo
- calendario de actividades
- fechas de evaluación
- lineamientos para uso de espacios
- información operativa para personal docente y coordinación

Desde el punto de vista de portafolio, esta evidencia permite mostrar experiencia en la construcción de interfaces estáticas orientadas a la **presentación de contenido web institucional**.

---

## Características principales

- Desarrollo con **HTML5** y **CSS3**
- Estructura semántica y ordenada
- Diseño visual con identidad institucional simulada
- Navegación principal con enfoque de portal
- Submenú de acceso rápido a secciones internas
- Botón flotante para volver al inicio
- Integración de imagen institucional como recurso visual
- Compatibilidad visual con escritorio y dispositivos móviles
- Organización de archivos con estructura limpia y profesional
